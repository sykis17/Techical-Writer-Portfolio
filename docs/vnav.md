# VNAV Instrument Approach Profile

This profile outlines the standard procedures for a Constant Descent Final Approach (CDFA) using VNAV, based on the RYR 2014 QRH standards.

---

## Profile Diagram
![VNAV Approach Profile](files/assets/diagrams/VNAVCHART.png)

---

### Procedural Overview
* **VNAV PTH:** Target mode for the final approach segment.
* **Speed Constraints:** Automatically managed by the FMC based on flap schedule.
* **Vertical Path:** Ensures a stabilized descent to the MDA/DA.
